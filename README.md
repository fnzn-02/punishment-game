# 🎮 벌칙자 게임

친구들과 함께 즐기는 벌칙자 결정 게임 모음입니다.
별도 설치 없이 브라우저에서 바로 플레이하세요.

🔗 **[지금 바로 플레이하기](https://fnzn-02.github.io/punishment-game/)**

---

## 🕹️ 게임 목록

| 게임 | 설명 |
|------|------|
| 🪜 **사다리타기** | 사다리를 타고 내려가 결과를 확인! 2~8명 |
| 🎡 **룰렛 돌리기** | 룰렛을 돌려 걸리는 사람이 벌칙 |
| 🎟️ **제비뽑기** | 접힌 제비 중 하나를 뽑아라 |
| 🔢 **업다운** | 숫자를 맞혀라, 못 맞히면 벌칙 |
| 💣 **벌칙캡슐 뽑기** | 캡슐 중 숨겨진 벌칙을 뽑으면 탈락 |
| ⚡ **반응속도 게임** | 신호가 뜨면 최대한 빨리 터치! 가장 느린 사람이 벌칙 |
| 🎯 **타이밍 게임** | 빠르게 움직이는 바늘을 초록 구역에 멈춰라 |
| ⏱️ **스탑워치 게임** | 화면 보지 말고 목표 시간에 딱 멈추기 |
