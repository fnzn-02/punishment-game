# 🎮 벌칙자 게임

친구들과 함께 즐기는 벌칙자 결정 게임 모음입니다.
별도 설치 없이 브라우저에서 바로 플레이하세요.

🔗 **[지금 바로 플레이하기](https://fnzn-02.github.io/punishment-game/)**

---

## 🕹️ 게임 목록

### 🎲 랜덤 게임 — 운에 맡겨라

| 게임 | 설명 |
|------|------|
| 🪜 **사다리타기** | 사다리를 타고 내려가 결과를 확인 (2~8명) |
| 🎡 **룰렛 돌리기** | 룰렛을 돌려 걸리는 사람이 벌칙 |
| 🎟️ **제비뽑기** | 접힌 제비 중 하나를 뽑아라 |
| 💣 **벌칙캡슐 뽑기** | 캡슐 중 숨겨진 벌칙을 뽑으면 탈락 |
| 🎲 **주사위 대결** | 3D 주사위를 굴려 순위 결정, 동점 시 재대결 |

### ⚔️ 대결 게임 — 실력으로 승부

| 게임 | 설명 |
|------|------|
| 🔢 **업다운** | 숫자를 맞혀라, 못 맞히면 벌칙 |
| ⚡ **반응속도 게임** | 신호가 뜨면 최대한 빨리 터치, 가장 느린 사람이 벌칙 |
| 🎯 **타이밍 게임** | 빠르게 움직이는 바늘을 초록 구역에 멈춰라 |
| ⏱️ **스탑워치 게임** | 화면 보지 말고 목표 시간에 딱 멈추기 |
| 🐹 **두더지 잡기** | 제한 시간 안에 두더지를 가장 많이 잡아라 |

## 📱 PWA — 앱처럼 설치하기

브라우저 주소창 없이 앱처럼 사용할 수 있습니다.

**iOS (사파리)**
1. 사파리에서 사이트 열기
2. 하단 공유 버튼 탭
3. "홈 화면에 추가" 선택

**안드로이드 (크롬)**
1. 크롬에서 사이트 열기
2. 우상단 점 3개 메뉴
3. "앱 설치" 또는 "홈 화면에 추가" 선택

## 🌙 기능

- 다크 / 라이트 모드 지원
- 모바일 최적화 (터치 지원)
- PWA 지원 — 홈 화면에 추가해 앱처럼 사용 가능
- 오프라인 지원 (서비스 워커 캐시)
- 설치 불필요 — 순수 HTML / CSS / JS
